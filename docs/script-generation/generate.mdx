The Script Generation endpoint allows you to create compelling video scripts using AI, with the ability to incorporate content from web sources and extract relevant images.

## Quick Start

```bash
curl -X POST "https://api.hoox.video/v1/script/generate" \
  -H "Authorization: Bearer hx_live_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Create a script about renewable energy innovations",
    "duration": 90
  }'
```

## Parameters

<ParamField body="prompt" type="string" required>
  The main instruction for script generation. Be specific about your requirements, target audience, and desired content.
</ParamField>

<ParamField body="duration" type="number" default={60}>
  Script duration in seconds (10-600). The AI will generate content to match this duration.
</ParamField>

<ParamField body="urls" type="array">
  URLs to extract content from (max 10). Content will be analyzed and incorporated into the script.
</ParamField>

<ParamField body="webSearch" type="boolean" default={false}>
  Enable enhanced web search capabilities for richer content generation.
</ParamField>

## Response Format

```json
{
  "data": {
    "script": "Welcome to our exploration of renewable energy...",
    "cost": 0.3,
    "extractedImages": [
      "https://example.com/solar-panel.jpg",
      "https://example.com/wind-turbine.jpg"
    ]
  }
}
```

## Integration Tips

### Content Strategy
1. **Prompt Engineering**: Use specific, detailed prompts for better results
2. **Source Quality**: Choose authoritative, recent sources for URL content
3. **Duration Matching**: Align duration with your platform requirements
4. **Cost Optimization**: Balance URL count with information needs

### Technical Implementation
```javascript
async function generateScript(prompt, options = {}) {
  const response = await fetch('https://api.hoox.video/v1/script/generate', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${process.env.HOOX_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      prompt,
      duration: options.duration || 60,
      urls: options.urls || [],
      webSearch: options.webSearch || false
    })
  });

  if (!response.ok) {
    throw new Error(`Script generation failed: ${response.status}`);
  }

  const { data } = await response.json();
  return data;
}

// Usage
const script = await generateScript(
  "Create a script about sustainable fashion",
  { 
    duration: 90, 
    urls: ["https://sustainable-fashion-blog.com/latest"],
    webSearch: true 
  }
);
```
