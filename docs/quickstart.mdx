## API Workflow Overview

The Hoox API follows a simple workflow to create professional videos:

<Steps>
  <Step title="Get Resources">
    Discover available voices and avatars for your videos
  </Step>
  <Step title="Generate Content">
    Create a script or directly generate a video from a prompt
  </Step>
  <Step title="Monitor Progress">
    Track generation status until completion
  </Step>
  <Step title="Export Video">
    Export your video to downloadable MP4 format
  </Step>
</Steps>

## Step 1: Get your API key

1. Log in to your [Hoox dashboard](https://app.hoox.video)
2. Go to **Settings** > **API**
3. Click **Generate API Key**
4. **Copy and save** your key (it will only be shown once)

<Warning>
  Keep your API key secret! Never share it publicly or commit it to version control.
</Warning>

## Step 2: Discover Available Resources

Before creating videos, explore the voices and avatars available for your account:

### Get Available Voices

<CodeGroup>

```bash cURL
curl -X GET "https://app.hoox.video/api/public/v1/resources/voices" \
  -H "Authorization: Bearer your_api_key"
```

```python Python
import requests

response = requests.get(
    'https://app.hoox.video/api/public/v1/resources/voices',
    headers={'Authorization': 'Bearer your_api_key'}
)

voices = response.json()['voices']
print(f"Found {len(voices)} voices")

# Find a professional English voice
professional_voices = [
    voice for voice in voices 
    if voice['language'] == 'en' and 'professional' in voice['tags']
]

selected_voice = professional_voices[0] if professional_voices else voices[0]
print(f"Selected voice: {selected_voice['name']} ({selected_voice['id']})")
```

```javascript Node.js
const response = await fetch('https://app.hoox.video/api/public/v1/resources/voices', {
  headers: {
    'Authorization': 'Bearer your_api_key'
  }
});

const data = await response.json();
const voices = data.voices;

// Find a professional English voice
const selectedVoice = voices.find(voice => 
  voice.language === 'en' && voice.tags.includes('professional')
) || voices[0];

console.log(`Selected voice: ${selectedVoice.name} (${selectedVoice.id})`);
```

</CodeGroup>

## Step 3: Generate Your First Video

You can generate videos in two ways: directly from a prompt, or by first generating a script for review.

### Option A: Direct Video Generation

<CodeGroup>

```bash cURL
curl -X POST "https://app.hoox.video/api/public/v1/generation/start" \
  -H "Authorization: Bearer your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Create an explanatory video about sustainable energy solutions",
    "duration": 60,
    "voice_id": "voice_en_us_female_sarah",
    "avatar_id": "avatar_business_woman_office_1",
    "format": "vertical",
    "web_search": {
      "script": true,
      "images": true
    },
    "animate_image": true
  }'
```


Poll the status endpoint until generation is complete:

<CodeGroup>

```bash cURL
curl -X GET "https://app.hoox.video/api/public/v1/generation/status/{job_id}" \
  -H "Authorization: Bearer your_api_key"
```

```python Python
import time

def wait_for_generation_completion(job_id, max_wait=600):
    """Wait for video generation to complete"""
    start_time = time.time()
    
    while time.time() - start_time < max_wait:
        status_response = requests.get(
            f'https://app.hoox.video/api/public/v1/generation/status/{job_id}',
            headers={'Authorization': 'Bearer your_api_key'}
        )
        
        status_data = status_response.json()
        status = status_data['status']
        progress = status_data.get('progress_step', 0)
        current_step = status_data.get('current_step', 'Processing')
        
        print(f"Status: {status} ({progress}%) - {current_step}")
        
        if status == 'completed':
            video_id = status_data['result']['video_id']
            cost = status_data['result']['cost']
            print(f"‚úÖ Generation completed! Video ID: {video_id}, Cost: {cost} credits")
            return video_id
        elif status == 'failed':
            error = status_data.get('error', {})
            print(f"‚ùå Generation failed: {error.get('message', 'Unknown error')}")
            return None
        
        time.sleep(10)  # Wait 10 seconds before checking again
    
    print("‚è∞ Generation timed out")
    return None

# Usage
video_id = wait_for_generation_completion(job_id)
```

```javascript Node.js
async function waitForGenerationCompletion(jobId, maxWait = 600000) {
  const startTime = Date.now();
  
  while (Date.now() - startTime < maxWait) {
    const statusResponse = await fetch(
      `https://app.hoox.video/api/public/v1/generation/status/${job_id}`,
      {
        headers: {
          'Authorization': 'Bearer your_api_key'
        }
      }
    );
    
    const statusData = await statusResponse.json();
    const status = statusData.status;
    const progress = statusData.progress_step || 0;
    const currentStep = statusData.current_step || 'Processing';
    
    console.log(`Status: ${status} (${progress}%) - ${currentStep}`);
    
    if (status === 'completed') {
      const videoId = statusData.result.video_id;
      const cost = statusData.result.cost;
      console.log(`‚úÖ Generation completed! Video ID: ${videoId}, Cost: ${cost} credits`);
      return videoId;
    } else if (status === 'failed') {
      const error = statusData.error || {};
      console.log(`‚ùå Generation failed: ${error.message || 'Unknown error'}`);
      return null;
    }
    
    await new Promise(resolve => setTimeout(resolve, 10000)); // Wait 10 seconds
  }
  
  console.log('‚è∞ Generation timed out');
  return null;
}

// Usage
```

## Step 5: Export the Video

Export your video to a downloadable MP4 file:

<CodeGroup>

```bash cURL
curl -X POST "https://app.hoox.video/api/public/v1/export/start" \
  -H "Authorization: Bearer your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "video_id": "YOUR_VIDEO_ID",
    "format": "vertical"
  }'
```

```python Python
def export_and_download_video(video_id, format="vertical"):
    """Export video and download the MP4 file"""
    
    # Start export
    export_response = requests.post(
        'https://app.hoox.video/api/public/v1/export/start',
        headers={
            'Authorization': 'Bearer your_api_key',
            'Content-Type': 'application/json'
        },
        json={
            'video_id': video_id,
            'format': format
        }
    )
    
    export_data = export_response.json()
    export_job_id = export_data['job_id']
    print(f"Export started! Job ID: {export_job_id}")
    
    # Wait for export completion
    while True:
        export_status_response = requests.get(
            f'https://app.hoox.video/api/public/v1/export/status/{job_id}',
            headers={'Authorization': 'Bearer your_api_key'}
        )
        
        export_status = export_status_response.json()
        status = export_status['status']
        progress = export_status.get('progress', 0)
        
        print(f"Export status: {status} ({progress}%)")
        
        if status == 'completed':
            video_url = export_status['result']['video_url']
            print(f"‚úÖ Export completed! Download URL: {video_url}")
            
            # Download the video
            video_response = requests.get(video_url)
            with open(f'video_{video_id}.mp4', 'wb') as f:
                f.write(video_response.content)
            
            print(f"üì• Video downloaded as video_{video_id}.mp4")
            return video_url
            
        elif status == 'failed':
            print(f"‚ùå Export failed: {export_status.get('error', {}).get('message', 'Unknown error')}")
            return None
        
        time.sleep(5)  # Wait 5 seconds before checking again

# Usage
if video_id:
    download_url = export_and_download_video(video_id)
```

## Complete Workflow Example

Here's a complete end-to-end example that puts it all together:

```python
import requests
import time

class HooxVideoCreator:
    def __init__(self, api_key):
        self.api_key = api_key
        self.base_url = 'https://app.hoox.video/api/public/v1'
        self.headers = {
            'Authorization': f'Bearer {api_key}',
            'Content-Type': 'application/json'
        }
    
    def create_video_from_prompt(self, prompt, duration=60):
        """Complete workflow: prompt -> script -> video -> export -> download"""
        print("üöÄ Starting video creation workflow...")
        
        # Step 1: Get resources
        voices = self.get_voices()
        avatars = self.get_avatars()
        
        selected_voice = next((v for v in voices if v['language'] == 'en' and 'professional' in v['tags']), voices[0])
        selected_avatar = next((a for a in avatars if 'business' in a['tags']), avatars[0])
        
        print(f"üì¢ Selected voice: {selected_voice['name']}")
        print(f"üë§ Selected avatar: {selected_avatar['name']}")
        
        # Step 2: Generate script
        script_data = self.generate_script(prompt, duration)
        script = script_data['script']
        
        print(f"üìù Script generated (cost: {script_data['cost']} credits)")
        print(f"Preview: {script[:100]}...")
        
        # Step 3: Generate video
        video_id = self.generate_video(script, selected_voice['id'], selected_avatar['id'])
        
        if not video_id:
            print("‚ùå Video generation failed")
            return None
        
        # Step 4: Export video
        download_url = self.export_video(video_id)
        
        print(f"‚úÖ Complete! Video ready at: {download_url}")
        return download_url
    
    def get_voices(self):
        response = requests.get(f"{self.base_url}/resources/voices", headers=self.headers)
        return response.json()['voices']
    
    def get_avatars(self):
        response = requests.get(f"{self.base_url}/resources/avatars", headers=self.headers)
        return response.json()['avatars']
    
    def generate_script(self, prompt, duration):
        response = requests.post(
            f"{self.base_url}/script/generate",
            headers=self.headers,
            json={'prompt': prompt, 'duration': duration, 'webSearch': True}
        )
        return response.json()['data']
    
    def generate_video(self, script, voice_id, avatar_id):
        response = requests.post(
            f"{self.base_url}/generation/start",
            headers=self.headers,
            json={
                'script': script,
                'voice_id': voice_id,
                'avatar_id': avatar_id,
                'format': 'vertical',
                'animate_image': True
            }
        )
        
        job_id = response.json()['job_id']
        return self.wait_for_completion(job_id, 'generation')
    
    def wait_for_completion(self, job_id, job_type):
        while True:
            response = requests.get(
                f"{self.base_url}/{job_type}/status/{job_id}",
                headers=self.headers
            )
            
            data = response.json()
            status = data['status']
            
            if status == 'completed':
                if job_type == 'generation':
                    return data['result']['video_id']
                else:
                    return data['result']['video_url']
            elif status == 'failed':
                print(f"‚ùå {job_type} failed: {data.get('error', {}).get('message', 'Unknown error')}")
                return None
            
            time.sleep(10)
    
    def export_video(self, video_id):
        response = requests.post(
            f"{self.base_url}/export/start",
            headers=self.headers,
            json={'video_id': video_id, 'format': 'vertical'}
        )
        
        export_job_id = response.json()['job_id']
        return self.wait_for_completion(export_job_id, 'export')

# Usage
creator = HooxVideoCreator("your_api_key")
video_url = creator.create_video_from_prompt(
    "Create an engaging video about the benefits of remote work for modern companies",
    duration=90
)
```

## Understanding Costs

Your first video will likely cost around **60-80 credits**:

- **Script generation**: 15 credits (with web search)
- **Base video (60s)**: 60 credits
- **Avatar usage**: +20 credits
- **Image animation**: +10-25 credits per image
- **Export**: FREE for API videos

<Info>
**Free Credits**: New accounts receive 100 free credits to get started. This is enough to create 1-2 complete videos and test the platform.
</Info>

## Workflow Optimization Tips

<AccordionGroup>
  <Accordion title="Use Resource Discovery">
    Always check available voices and avatars first. This ensures you're using IDs that exist in your plan and helps you choose the best options for your content.
  </Accordion>
  
  <Accordion title="Generate Scripts First">
    Generate scripts separately before video creation. This allows you to:
    - Review and edit content before committing to full generation
    - Get accurate cost estimates
    - Ensure quality before spending credits
  </Accordion>
  
  <Accordion title="Monitor Status Efficiently">
    - Start with 5-10 second polling intervals
    - Increase intervals as time progresses
    - Use webhooks for production applications
    - Always handle timeout scenarios
  </Accordion>
  
  <Accordion title="Optimize Export Strategy">
    - API videos have free exports - use this advantage
    - Export multiple formats if needed (vertical, square, landscape)
    - Download videos immediately - URLs expire after 24 hours
  </Accordion>
</AccordionGroup>

## Production Considerations

When building production applications:

1. **Error Handling**: Implement robust retry logic and error handling
2. **Rate Limiting**: Respect API rate limits and implement backoff strategies
3. **Webhooks**: Use webhooks instead of polling for better efficiency
4. **Caching**: Cache resource lists (voices/avatars) to reduce API calls
5. **Monitoring**: Track credit usage and set up alerts for low balances

<Tip>
**Best Practice**: Start with script generation to validate content and costs before committing to full video generation. This approach saves credits and ensures better results.
</Tip>
